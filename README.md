# Gesture-controlled-robot

This project implements a gesture controlled robot using Arduino, MPU6050 accelerometer and nRF24L01 wireless module. The robot moves based on hand gestures detected by the accelerometer.

 # Features
* Hand gesture control
* Wireless communication
* Real-time robot movement
* Arduino based implementation

# Components Used
* Arduino Uno / Nano
* MPU6050 Accelerometer
* nRF24L01 Wireless Module
* L298N Motor Driver
* DC Motors
* Robot Chassis

# Working Principle
Hand gestures are detected using an MPU6050 accelerometer. The Arduino processes these values and sends commands wirelessly to the robot. The receiver Arduino controls the motor driver to move the robot.

## Applications
* Human machine interaction
* Robotics research
* Assistive robotics
* Industrial robot control
